# docker-nettools
 
 A simple, small, alpine-based Docker image with some handy networking tools
installed:

- net-tools
- conntrack-tools
- coreutils
- curl
- wget
- drill
- iperf3
- iproute2
- iptables
- iputils
- ip6tables
- keepalived
- nftables
- socat
- netcat-openbsd
- bind-tools
- ethtool
- mtr
- tcpdump
- lshw

Please see the Dockerfile for a complete list of tools.
